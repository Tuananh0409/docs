# SOFTWARE REQUIREMENTS SPECIFICATION (SRS)

## Phát triển phần mềm quản lý dự án nội bộ

# I. Tổng quan tài liệu

## 1. Mục đích tài liệu

Hệ thống được thiết kế phục vụ cho toàn bộ nhân sự trong công ty.

## 2. Phạm vi tài liệu

-   Quản lý Workspace
-   Quản lý Project
-   Quản lý Task

## 3. Đối tượng sử dụng

Toàn bộ nhân sự công ty.

## 4. Vai trò người dùng

-   Admin
-   Manager
-   Lead
-   User
-   Stakeholder
